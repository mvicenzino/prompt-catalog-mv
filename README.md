# Prompt Catalog App

A premium, dark-themed application to organize and discover prompts for AI generation (Midjourney, ChatGPT, etc.).

## Features
- **Dashboard**: View all your prompts in a responsive grid.
- **Categories**: Filter by Photos, Apps, Images, Coding, Writing.
- **Search**: Real-time search by title, content, or tags.
- **Add Prompt**: Easily add new prompts with source tracking (X, Reddit, etc.).
- **Local Storage**: Your prompts are saved to your browser automatically.

## Tech Stack
- React + Vite
- Vanilla CSS (Premium Dark Theme)
- Lucide React Icons
- React Router

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) in your browser.
